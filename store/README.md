# store/

Store submission assets for stikfix. All copy in this folder is ready to paste into the respective developer consoles.

## Contents

| Path | Description |
|---|---|
| `chrome-web-store/listing.md` | All CWS listing fields: name, summary, detailed description, keywords, reviewer notes |
| `chrome-web-store/permission-justifications.md` | CWS permission justifications, single purpose statement, remote code answer, data use disclosures |
| `chrome-web-store/screenshots/` | Store screenshots (1280×800 PNG) — created by assets agent |
| `firefox-amo/listing.md` | All AMO listing fields: name, summary, about, tags, license, source code note, reviewer notes |
| `firefox-amo/screenshots/` | AMO screenshots — created by assets agent |
| `promo/` | Promo tiles: 440×280 (CWS required), 920×680 (CWS optional) — created by assets agent |
| `assets-src/` | Source files for screenshots and promo tiles — created by assets agent |
| `privacy-policy.md` | Hostable privacy policy — reference from both store listings |
| `SUBMISSION-CHECKLIST.md` | Step-by-step packaging and upload checklist for CWS and AMO |
